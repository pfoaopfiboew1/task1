автоматизированная система для базового аудита безопасности сервера.
проект проверяет открытые порты, собирает информацию об ос и формирует отчет.

## инструкция_по_запуску

требуется python 3.6+.

1. **клонирование**:
   ```bash
   git clone https://github.com/pfoaopfiboew1/task1.git
   ```

2. **запуск**:
   ```bash
   python3 security_monitor.py
   ```

## результаты_работы

скрипт выводит данные в консоль и сохраняет их в `security_report.txt`.
